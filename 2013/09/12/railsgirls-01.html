Rails Girls #01: Say Hello to Rails, Girls :>

Sep 12, 2013

離Rails Girls結束已經有好一段時間，稍稍拖了一下才來寫心得。很開心有機會可以參加第一屆的Rails Girls，當Coach的經驗很有趣，從一開始參與討論，設計課程到抽學員到活動開始那天，其實Coach的準備沒有想像中多，但這是因為經驗豐富的Organizers @龍哥、@慕凡和@泰安規劃召集才有這麼好的景況XD

Rails Girls 的報名在開始上線的兩天內就很快地關閉掉截止了，因為其實 CLBL 可容納的人數最多也就能有25個學員，然後藉助了各方的宣傳（主力可能是Inside）整個報名狀況完全是出乎一開始所有宅宅教練的預期 – 怕人數不足，兩天內衝到 325 人，從三百多位佳麗(XD)中選出 25 位真的是太難了，所以緊急關閉報名，導致了粉絲團頁面上出現了好多正妹的殘念。本來 RailsGirls 的定調就是希望女生能夠藉助這樣的活動體驗 coding 的樂趣，所以「初學者」或是不曾有過相關經驗的女生會是主要的客層（？），因為定調如此，本來很擔心人數會不足的幾位召集人，沒有想到竟然報名會爆衝… 這其實也說明了一個事實： 女生真的不是看到數學工程和程式就怕，有興趣的女孩子還是非常多的，而且以報名者填寫的原因來說，真的想要學著做一些東西的需求在女學員報名者中是一個很重要的報名動力。

環境啊…

我個人覺得架 Ruby on Rails 環境一向就是最麻煩的事情，就像是@Rack lin阿土伯之前說過的：「Rails 開發者的跨平台就是～從 Mac 上面 deploy 到 Ubuntu 伺服器」，也就是說，Rails 根本一點也不在乎所謂跨平台的事情XD

但是為了怕初學者因為其實很崩潰的架環境過程失去興趣，從一開始的討論就很在意如何快快幫學員把環境架好可以開始 rails generate scaffold …(誤)

統計了學員會帶來的作業系統以後，就擬定了三個劇本預備快速裝好環境，還有各作業系統的「雨備」（當真的卡關的時候可以快速跳過這步驟的 VM 和預備機器），結果出乎意料的是後來參與的學員有很多人都將行前提醒事項上面的預備安裝準備好了，甚至有早就裝好所有環境的… (像我的學員)，但是當初為了預防萬一在這個步驟給了一個晚上又半個早上的時間，事實證明還是多少需要的。

但是總之，事實證明來參加的學員們學東西是很認真的！

Rails Girls 的美味便當盒

制作一個網站其實用到了很多技術，怎麼想都覺得不太可能在短短一天多的課程內完成，所以規劃了很簡單的「便當盒」，盒子裡的每一格都放了一大類的素材，而學員在第一階段要做的就是知道這些素材該放在哪一個格子裡。

便當盒每一格都代表了要放進同一類的「食材」，包含了製作一個網站需要的資料儲存空間（Storage），網站背後運作的邏輯（Logic）和網站外皮（Style and Structure），當然還有一些架伺服器的基本設施（Infrastructure），投影幕上放了一堆食材等著學員將之放入相對應的格子內…

這個階段是本次唯一有同步教學的小活動，最主要的目的只是讓學員可以基本的認識製作一個網站會用到的工具組合，每一格裡面的食材在世界上都有很多種類可以選擇，但是每個格子都不能空下來，才能組成一個美味便當盒。

實作時間

其實 Rails Girls 並不是著重於把學員真正的教會（一天半的時間而已假如說真的可以教會也太唬爛了XD），最大的目的是在於讓女孩子也可以親身體驗寫程式的樂趣，Rails 的其中一個特色功能就是一行 command line 就可以 generate 一個「鷹架」(scaffold)，很快速的生成一個最基本的 CRUD (新增、顯示、修改、刪除) 架構製作出一個 Web form，在 Rails 4 中甚至可以直接產生關聯式資料表的scaffold，對新手來說就是一件成就感++的過程（XD）。由於一開始報名階段就已經挑選過參加者的資格，雖然說大部份的學員都是新手或不曾接觸 coding 的背景，其中有不少人原本就是前端或網站設計出身，所以程度之間還是有差異，這時候個人教練的配置是比較合適的，例如我的學員就完全可以直接跳過下 scaffold 的成就感++階段直接來硬派的 MVC 純手作。

只是在實作時間最後的發表眾教練都深感前端真的好威啊，同樣是個 scaffold，用精緻前端秀出成果的氣勢就是完全不同，一點都感覺不出是新手練習品…

相較起來，就算是靠實力手刻的 MVC code 沒有前端設計和直接 scaffold command 下下去呈現出來的成果根本沒有差異，後端工程師啊…（淚）

後記

因為專案轟炸和出國的關係拖了好久好久才把這篇草稿寫出來，但是回想起來整個過程還是很好玩，而且當 Rails Girls 的教練在每天被工作瘋狂攻擊的日子就像是一種紓壓啊（誤），希望十月份的 Rails Girls 能有新氣象囉 :)
