一天的小蜜月（記于 2014/01/01 ）

Jan 8, 2014

前言

幾分鐘前終於又忙完了一年的春節訂票，到現在才有真正稍微小小鬆ㄧ口氣的感覺。其實我很慶幸自己有幸可以接到這麼大的交通案，台灣能有這種固定瞬間爆量的網站不多，而且，業主的需求數十年如一日，只是太過於龐大複雜了，特例太多，再花十年也不一定做得完，只是從另一個角度看，這個案子就像是一個完美的軟體試驗場。

眼尖的朋友已經發現這個 blog 一到年底更新的頻率就很微弱，因為太忙啦XD

其他人都已經數完倒數計時整整 7 天了，現在我才終於喘口氣可以做一點自己的雜事，所以 2013 年 shesee 解開了哪些成就…

2013 年的成就

  1. 解開參加 WebConf 的成就
  2. 解開天瓏 VIP 的成就
  3. 解開第一年春節訂票的成就
  4. 解開接手 JSP 專案的成就
  5. 解開參加監察院會的成就
  6. 解開助擬官方新聞稿的成就
  7. 解開打掉 ATIS 即時交通資訊系統重做的成就
  8. 參加 @高見龍 的有心人活動的成就
  9. 解開探索馬來西亞的成就
  10. 解開在 Ruby Tuesday 講一場超小 talk 的成就
  11. 認識一群很可愛的強者的成就
  12. 解開在 Hitcon 吃哈根大使的成就
  13. 解開當龍哥自由軟體鑄造場 Git 課程當助教的成就
  14. 解開 Rails Girls 教練的成就
  15. 解開一個專案 commit 了 50 萬+ 行 code 的成就
  16. 解開去新加坡看都市與交通規劃的成就
  17. 解開自己坐公車跨越國境，去樂高樂園的成就
  18. 又再解開第二次 Rails Girls 教練的成就

應該不止這些，其實 2013 年倒數兩週我的人生陷入了前所未有的荒唐，從家裡的事業發生了一個危機，被某個議員賴帳開始，達成了更多的成就（誤）

  1. 第一次使用 蒐證雲 錄音
  2. 意外上了八卦板
  3. 解開被記者採訪的成就
  4. 解開被威脅要上法院的成就
  5. 解開 3 天寫出簡單服務買好 domain 就直接推上線的成就
  6. 解開在臺北車站的遊民和阿媽們眾目睽睽之下被求婚的成就

這些成就連番發生時間不到兩週，應該說密集起來只有大約七天的時間…六天的時間腦子高速運轉在想怎麼突破困境，焦躁到好幾天滴水滴食未進，疊合了最後一天的時間腦子呈現完全的空白 XD

大概求婚不管對甲方乙方都是種很衝動的行為，而大約也只有我的甲方會選擇在最難的時刻說出最難的承諾。對他來說，這只是兩個人的事情，不要任何的朋友一起熱鬧，也不需要熱鬧，餐廳總有一天會倒，所以選了我今年一直努力付出的臺北車站，選了跨越一生一世的一天。

關於那枚戒指

由徐曦留言。

跨年這天腦子真的是很空白的，完完全全，明明前一刻才心心念念著如何幫爸媽處理那些骯髒可憎的事，2014 年第一天早上，兩個人騎著腳踏車去了信義區的 T 店改戒圍，一路從士林，內湖，信義區，臺大，最後回到士林，陽光燦爛，好像是幾天前那些壓力和煩憂都是上輩子的事，講的都是只關乎兩個人之間的話題，人們鬧著跨年早上都在家裡補眠了，路上也只剩下我們，他會絮絮叨叨的說著鑽石的色澤和淨度，就像是一個活體保證書，並且要讓我知道以後一切都會和現狀一樣，聽著聽著原本的惶惑不安也就少一點了（相信我之術），我突然感覺，那些瑣事都不是很重要，我知道他總是要不惜一切給我最好的那個。

一直說到，認識的時候其實也才十七歲，那年的 11/28 半夜 12 點在台南，我的生日每次都逢期中考，夜裏大家念完書就端著蛋糕點好蠟燭在成大 K 館那個長長的階梯上替我慶生，他也在場，可是後來我一直都不知道也沒發現，參與的人也都不記得了 XD

然後一年一年過去，混雜了他的蹉跎多心和我的神經大條，年少也就這麼過去了。

只是今天他講起來我仍然很不可置信，原來那個時候就已經偷偷地走進我的人生，並且，看起來是要努力和我一起走到終點的那個。我總覺得人可以一直走下去，與其說是愛情，不如說是義氣，你看起來好像就是一副很有義氣的樣子啊 XD

四年蹉跎，兩年台南台北，一年當兵，一年外派，其實我擔心的只是自己已經野慣了，不知道要怎麼相處，害怕太靠近，害怕吵架，害怕改變，害怕太多未知的事情，所以那枚戒指我遲遲未能真正打開，等到 1 月 1 號早上才真的拿起來，真的很注意很仔細的看了那其中包含了多少你想讓我知道的美好。

我何其有幸可以一生中可以有機會只與一個人相愛，並且他的身上收藏了連我自己都忽略的年少記憶，也將乘載我的未來，雖然不知道是不是真的就可以像是日子的象徵一樣，跨越一生一世。2014 年的第一天，就像一天很短暫的蜜月，一邊走了長長的路，一邊回憶長長的年少，並且一邊看他非常緊張 Facebook 的按讚數 XD

人生苦短，又苦又短，有著生老病死的苦楚與磨難，也有像是費盡心思送出一個承諾的喜悅，也許這些苦樂一直都是並存的，我從不覺得求婚這件事情應該要狂喜，應該要感動流淚，我還在迷惑應該要給出怎麼樣的反應，或許也不應該想得太多，因為如果有被害妄想症如你都已經覺得沒有問題，我應該也可以 default it’s working.

只是這件事我非常確定，我真的很喜歡那天。

關於 2014

終於把 2013 最後一波砲火（春節訂票）也抵禦完成了，今年的新希望，就是可以更加的增加值得給 talk 和寫 note 的技術，並且想要好好的把手邊毫無章法的工作整理成有秩序的工作流程，想要親手建立一個 CI Server，想要把大量資料刷新的問題想出更好的解法。想要能在 Conf 中上台。

想要繼續維持住有話直說的自己，但是先努力更新成 1.1 版，比較委婉的那種。

想要，每天都把握著，好好生活。
